# Changelog

All notable changes to **Todo Breeze (PWA)** will be documented here.

## [0.1.1] - 2025-11-04
### Changed
- Improved the service worker update flow to auto-activate new releases without reloading the very first visit.
- Documented the deployment refresh process in the README.

## [0.1.0] - 2025-10-29
### Added
- Initial release: vanilla HTML/CSS/JS PWA
- Task CRUD with priorities, due dates, tags (#hashtags in title)
- Filters (tag/date), sorting, search
- Import/Export JSON
- Offline via service worker
- Dark/Light theme toggle
